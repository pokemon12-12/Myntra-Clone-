Myntra Clone (React)
A responsive e-commerce frontend clone of Myntra built using React. This project replicates core features of a modern fashion shopping platform, including product listing, filtering, cart management, and dynamic UI interactions. It demonstrates component-based architecture, state management, and reusable UI design patterns.

Tech Stack: React, React Router, Redux Toolkit, CSS/Styled Components (adjust if needed)

Key Features:

Home page with dynamic product listings
Product filtering and category browsing
Add to cart / remove from cart functionality
Responsive and mobile-friendly UI
State management using Redux Toolkit
Clean component-based architecture
